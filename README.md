# 🧮 Multi Button Counter App

A simple React app that lets you increase, decrease, and reset a counter using multiple buttons.

## 🚀 Features
- Increase count (+1)
- Decrease count (–1)
- Reset to zero
- Styled buttons with hover effect

## 🛠️ Built With
- React
- CSS (custom styles)
